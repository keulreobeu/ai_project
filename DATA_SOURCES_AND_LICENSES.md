# 데이터 출처 및 라이선스

최종 확인일: 2026-07-16

이 문서는 LocalHub 서비스가 사용하는 데이터의 원출처, 이용 조건, 표시 문구 및 운영상 주의사항을 정리한다. 법률 자문을 대신하지 않으며, 공개 배포 전에는 각 제공기관의 최신 조건을 다시 확인해야 한다.

## 1. 한국관광공사 TourAPI 4.0

| 항목 | 내용 |
|---|---|
| 제공기관 | 한국관광공사 |
| 공식 데이터명 | 한국관광공사_국문 관광정보 서비스_GW |
| API | TourAPI 4.0 / `KorService2` |
| 공식 페이지 | https://www.data.go.kr/data/15101578/openapi.do |
| 저장 위치 | `data/**.json`, `project/seoul_festival.db` |
| 주요 데이터 | 관광지, 문화시설, 축제·공연, 여행코스, 레포츠, 숙박, 쇼핑, 음식점 |
| 업데이트 주기 | 공식 API 기준 실시간 |
| 비용 | 무료 |
| 공식 이용허락범위 | 이용허락범위 제한 없음 |

공공데이터포털은 저작권 제약 없이 활용 가능한 관광정보를 선별해 API로 제공한다고 안내한다. 웹, 모바일 앱, 홍보물 등 다양한 매체에서 활용할 수 있다.

### 사진 및 이미지 주의사항

TourAPI의 텍스트·메타데이터와 사진을 동일한 라이선스로 단정하면 안 된다.

- 공식 페이지는 제공 사진에 공공누리 제1유형과 제3유형 이미지가 함께 포함된다고 고지한다.
- 제1유형은 출처 표시가 필요하다.
- 제3유형은 출처 표시와 변경 금지 조건이 적용된다.
- 현재 저장된 JSON의 이미지 URL만으로 각 이미지의 유형을 확정하기 어렵다.
- 개별 이미지 유형을 확인하지 못한 경우, 안전하게 출처를 표시하고 자르기·합성·색상 변경 등 이미지 변형을 피한다.
- 사진을 명예훼손, 인격권 침해, 사회 통념에 반하는 용도 또는 기업 CI·BI로 사용하지 않는다.
- 사진 속 인물, 상표, 건축물 등에 관한 초상권·상표권 등 제3자 권리는 별도로 확인한다.

공공누리 유형 설명: https://www.kogl.or.kr/info/license.do

### 권장 출처 표시

서비스 소개, 데이터 출처 또는 푸터에서 다음 문구를 표시한다.

```text
관광정보 및 이미지는 한국관광공사 TourAPI 4.0을 활용했습니다.
출처: 한국관광공사
https://www.data.go.kr/data/15101578/openapi.do
```

이미지의 개별 공공누리 유형을 확인할 수 있다면 해당 이미지 주변이나 상세 화면에 유형과 원출처를 함께 표시한다.

## 2. 저장소에 포함된 데이터 현황

실제 추적 중인 JSON의 `items` 배열을 기준으로 계산한 값이다.

| 지역 | JSON 파일 | 레코드 | 포함 유형 |
|---|---:|---:|---|
| 광주·전라권 | 8 | 1,393 | 8종 전체 |
| 구미·경북권 | 8 | 1,667 | 8종 전체 |
| 대전·충청권 | 8 | 1,365 | 8종 전체 |
| 부산 | 7 | 1,759 | 음식점 제외 7종 |
| 서울 | 7 | 6,518 | 음식점 제외 7종 |
| 합계 | 38 | 12,702 |  |

`project/seoul_festival.db`의 `places` 테이블에는 서울 데이터 6,518건이 들어 있으며, JSON과 동일한 TourAPI 파생 데이터다. 커뮤니티 게시글은 이 공공데이터에 포함되지 않는다.

## 3. OpenStreetMap 지도

| 항목 | 내용 |
|---|---|
| 데이터 제공 | OpenStreetMap contributors |
| 데이터 라이선스 | Open Data Commons Open Database License 1.0 (ODbL) |
| 저작권 안내 | https://www.openstreetmap.org/copyright |
| 타일 서버 | `https://tile.openstreetmap.org/{z}/{x}/{y}.png` |
| 타일 정책 | https://operations.osmfoundation.org/policies/tiles/ |

현재 프런트엔드는 지도 하단에 `© OpenStreetMap contributors` 링크를 표시한다. 이 표시는 제거하거나 가리면 안 된다.

표준 OSM 타일 서버는 무료 지도 데이터와 별개의 제한된 공용 서비스다. 대량 다운로드, 오프라인 저장, 타일 사전 수집을 하지 않고 브라우저 캐시와 Referer 정책을 유지해야 한다. 트래픽이 커지는 운영 서비스에서는 별도 타일 제공자 또는 자체 호스팅을 검토한다.

## 4. 사용자 작성 데이터

커뮤니티 게시글의 제목, 본문, 비밀번호는 사용자가 직접 입력한 데이터이며 TourAPI 라이선스 대상이 아니다.

- 게시글의 이용허락, 삭제, 신고 및 운영 정책을 서비스 약관에 별도로 명시한다.
- 게시글 원문 또는 일부가 챗봇 프롬프트로 외부 AI 제공자에게 전송될 수 있다면 개인정보 처리방침과 이용자 안내에 이를 표시한다.
- 비밀번호와 개인정보를 데이터 출처 표시나 챗봇 응답에 노출하지 않는다.

## 5. AI 생성 답변

Gemini 또는 다른 언어 모델의 답변은 원본 관광 데이터가 아니며, 한국관광공사의 공식 답변으로 표시하면 안 된다.

- 화면에 AI 생성 답변임을 알린다.
- 관광지·행사 정보는 TourAPI 원본 레코드를 출처로 함께 제시한다.
- 모델 제공자의 최신 API 약관, 데이터 처리 조건 및 표시 요구사항을 별도로 준수한다.

## 6. 공개 배포 전 체크리스트

- [ ] 서비스 푸터 또는 데이터 출처 화면에 한국관광공사 출처를 표시한다.
- [ ] 이미지별 공공누리 유형을 확인하거나, 확인 불가 시 이미지 변형을 피한다.
- [ ] OpenStreetMap attribution이 지도 위에서 항상 보이는지 확인한다.
- [ ] OSM 표준 타일을 대량 수집하거나 오프라인 저장하지 않는다.
- [ ] 데이터 수집일 또는 마지막 동기화 시각을 기록한다.
- [ ] 커뮤니티 약관과 개인정보 처리방침을 마련한다.
- [ ] AI 답변과 공식 관광정보를 시각적으로 구분한다.
- [ ] API 운영계정 심의 및 트래픽 한도를 확인한다.
